Виктор Гамов о Kafka Streams IQ на jug.msk.ru

Блог компании JUG Ru Group Java *
8 мая 2019 года на встрече московского сообщества Java-разработчиков jug.msk.ru Виктор Гамов рассказал про Kafka Streams IQ (Interactive Queries).



О докладчике


Виктор в четвёртый раз выступает на jug.msk.ru, предыдущие его доклады были про Nashorn (13 ноября 2014 года), Hazelcast (4 мая 2016 года), Apache Kafka (23 апреля 2018 года). Как и год назад, в данный момент работает в компании Confluent, но сейчас уже в качестве Developer Advocate.

Сооснователь и ведущий популярного подкаста Разбор полётов, соавтор книги Enterprise Web Development, спикер многих конференций (в том числе организуемых JUG.ru Group). Сайт Виктора (с техническим блогом там же), видео докладов на YouTube, файлы презентаций на Speaker Deck и Notist, Хабр, Twitter и GitHub.

Некоторые доклады, сделанные с момента прошлого выступления на jug.msk.ru:

  • «Kafka на Kubernetes: загадочная история создания облачной платформы поточной обработки данных» (DevOops 2018: презентация)
  • «Java Puzzlers NG S03: What? Where Did Those Come From?!» (с Барухом Садогурским на Oracle Code One 2018: видео)
  • «Fight Crime with Kafka Streams and the Bintray Firehose API» (с Барухом Садогурским на Oracle Code One 2018: видео, презентация)
  • «Crossing the Streams: Rethinking Stream Processing with KStreams and KSQL» (DevNexus 2019: презентация)
  • «Kafka on Kubernetes: Does it really have to be “The Hard Way”?» (Kafka Summit 2019: презентация)

О докладе


Традиционно перед представлением докладчика и началом встречи Андрей Когунь напоминает о ближайших событиях — открыт приём заявок на Java-конференцию Joker (Санкт-Петербург, 25-26 октября) и ожидается проведение во второй раз IT-фестиваля TechTrain (Санкт-Петербург, 24-25 августа).



Доклад был посвящён Interactive Queries, составной части API Kafka Streams, предоставляющей возможность доступа к промежуточному состоянию приложения и полученным результатам без использования традиционных баз данных и хранилищ.



Показ слайдов презентации чередовался с демонстрацией примеров кода. Было задано большое количество вопросов, на которые Виктор моментально дал исчерпывающие ответы, что подтверждают и отзывы в Twitter.



Дискуссия продолжалась как в перерыве встречи, так и после неё. На встрече присутствовал один из спикеров конференций JUG.ru Иван Пономарёв, чей недавний доклад на конференции JPoint 2019 «Kafka Streams API: шаг за рамки Hello World» (презентация, видео пока доступно только участникам конференции) может дополнить рассказанное Виктором.
